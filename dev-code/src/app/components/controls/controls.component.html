Controls

Controls allow users to take actions, and make choices, with a single tap
  • Checkboxes
  • A. Checkboxes allow users to select one or more items from a set. Checkboxes can turn an option on or off.
  • Radio buttons
  • B. Radio buttons allow users to select one option from a set.
  • Switches
  • C. Switches toggle the state of a single item on or off.
